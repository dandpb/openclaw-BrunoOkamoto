# 🎓 OpenClaw by Bruno Okamoto — Complete Course Kit

**From zero to personal agent with OpenClaw.**

> How I built an AI assistant that replaced 3-4 people in 13 days — and how you can do the same.

By [Bruno Okamoto](https://www.youtube.com/@microsaas) 🍇

---

## 🚀 How to use this repository

1. **Watch the lesson** on the platform (Last Link)
2. **Find the material** in the table below — each lesson has a PDF, HTML and prompt
3. **Paste the prompt into your OpenClaw** — the agent guides you through the rest
4. **Use the use cases** to apply them to your daily routine
