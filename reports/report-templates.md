# Report Templates

> Design references for reports the agent can generate.

## 📊 Report Types

### 1. Weekly Metrics
- **Use:** Weekly summary of business metrics
- **Format:** Dark mode PDF, charts, comparison with previous week
- **Data:** MRR, churn, new customers, social media metrics
- **Example:** Amora's ChartMogul reports

### 2. Social Media Report
- **Use:** Social media performance
- **Format:** PDF with cards per platform
- **Data:** Followers, engagement rate, top posts, growth
- **Platforms:** Instagram, YouTube, LinkedIn, X/Twitter

### 3. Sponsor Report
- **Use:** Media kit / proposal for sponsors
- **Format:** Premium PDF (dark mode, hero stats, grid)
- **Data:** Audience, demographics, use cases, pricing
- **Example:** Amora's sponsor report (7 pages, premium design)

### 4. QA Report
- **Use:** Project quality audit
- **Format:** Markdown → PDF
- **Data:** Bugs found, severity, status, recommendations

### 5. Competitive Intelligence
- **Use:** Competitor analysis
- **Format:** PDF with SWOT, comparisons
- **Data:** Features, pricing, positioning, gaps

## 🎨 Design Guidelines

### For HTML → PDF reports:
```css
/* Premium dark mode */
body { background: #0d1117; color: #c9d1d9; }
.card { background: #161b22; border: 1px solid #30363d; border-radius: 8px; }
.stat { color: #58a6ff; font-size: 2rem; font-weight: bold; }
.label { color: #8b949e; font-size: 0.85rem; }

/* Accent colors */
.positive { color: #3fb950; }
.negative { color: #f85149; }
.neutral { color: #d29922; }
```

### Standard structure:
1. **Header:** Logo/name + period + generation date
2. **Hero stats:** 3-4 highlight metrics (large numbers)
3. **Sections:** Cards with detailed data
4. **Footer:** Data source + "Generated by [Agent Name]"

## 💡 Tip

To generate beautiful PDFs without installing LaTeX:
1. Agent generates styled HTML
2. Opens in browser (Playwright/Puppeteer)
3. Exports as PDF via browser

Works on any server without heavy dependencies.
